# Trackimo Desktop

League of Legends Draft Selection Desktop Application built with Tauri and React.

## Development

### Prerequisites

- [Rust](https://www.rust-lang.org/tools/install)
- [Node.js](https://nodejs.org/) (v16 or higher)
- [Tauri CLI](https://tauri.app/v1/guides/getting-started/prerequisites)

### Setup

1. Install dependencies:
```bash
npm install
```

2. Run in development mode:
```bash
npm run tauri dev
```

### Building

```bash
npm run tauri build
```

## Features

- LCU (League Client Update) API integration
- Real-time draft selection monitoring
- Machine learning powered recommendations (coming soon)
